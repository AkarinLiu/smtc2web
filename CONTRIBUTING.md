# 贡献指南

感谢您参与该项目的开发！

## 如何贡献

1. Fork 仓库到您的 GitHub 账户。
2. 克隆您的 fork 到本地。
3. 创建一个新的分支用于您的更改。
4. 在您的分支中进行更改。
5. 提交您的更改并推送至您的 fork。
6. 在 GitHub 上创建一个 Pull Request。

## Maintainer 权限给予策略

以下情况将给予 Maintainer 权限：

- 对项目做过重大贡献（例如修复恶意漏洞、将该软件移植至其他平台等）
- 对项目有长期贡献（例如持续维护、提供技术支持等）
- 对项目有特殊贡献（例如提供关键代码、提出重要建议等）
